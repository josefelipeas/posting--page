# Postagem de blog

Página que simula a criação de posts para a rede social.

## Descrição

Página criada para simular postagem em sites como Blogs ou Redes sociais. Utilizando formulário para receber informações do usuário, contendo título e conteúdo da postagem. Enviando essas informações para uma API externa e renderizando-as na página.

## Execução

Projeto executado com o programa VSCODE utilizando a extensão Live Server.

## Tecnologias utilizadas

Projeto desenvolvido com HTML, CSS e JavaScript.
